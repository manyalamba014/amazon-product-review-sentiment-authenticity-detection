# Amazon Product Review Sentiment & Authenticity Detection

## Overview
This project uses Natural Language Processing (NLP) and machine learning techniques to analyze Amazon product reviews and identify patterns associated with review authenticity and sentiment.

## Objectives
- Explore and visualize Amazon review data.
- Perform text preprocessing and feature engineering.
- Apply sentiment analysis techniques.
- Train and evaluate machine learning models for review classification.
- Compare model performance using standard evaluation metrics.

## Methods
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- TF-IDF Vectorization
- Sentiment Analysis
- Machine Learning Classification
- Model Evaluation and Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- NLTK
- Matplotlib
- Seaborn
- Jupyter Notebook
